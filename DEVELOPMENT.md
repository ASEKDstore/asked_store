# Руководство по разработке

## Начало работы

### 1. Установка зависимостей

```bash
pnpm install
```

### 2. Настройка базы данных

1. Создайте PostgreSQL базу данных
2. Скопируйте `.env.example` в `apps/api/.env` и заполните переменные
3. Запустите миграции:

```bash
pnpm db:migrate
```

4. (Опционально) Заполните тестовыми данными:

```bash
pnpm db:seed
```

### 3. Запуск в режиме разработки

Запустить все сервисы одновременно:
```bash
pnpm dev
```

Или по отдельности:
```bash
# Админ панель (порт 3001)
pnpm dev:admin

# Мини-приложение (порт 3002)
pnpm dev:miniapp

# API сервер (порт 3000)
pnpm dev:api

# Telegram бот
pnpm dev:bot
```

## Структура проекта

### apps/admin
Админ панель на Next.js для управления:
- Товарами
- Баннерами
- Страницами
- Заказами
- Статистикой

### apps/miniapp
Мини-приложение для пользователей Telegram:
- Главная страница
- Каталог товаров
- Карточка товара
- Корзина
- Профиль пользователя

### apps/api
Backend API на Fastify:
- REST API для админки
- REST API для мини-приложения
- Аутентификация
- Загрузка файлов

### apps/bot
Telegram бот:
- Команды бота
- Интеграция с мини-приложением
- Уведомления

## Переменные окружения

### apps/api/.env
```
DATABASE_URL="postgresql://user:password@localhost:5432/telegram_shop"
JWT_SECRET="your-secret-key"
PORT=3000
```

### apps/bot/.env
```
TELEGRAM_BOT_TOKEN="your-bot-token"
MINI_APP_URL="https://your-miniapp-url.com"
```
